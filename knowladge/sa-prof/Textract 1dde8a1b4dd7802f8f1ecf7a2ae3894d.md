# Textract

## **Amazon Textract - Key Concepts**

Amazon Textract is an AWS service that utilizes artificial intelligence (AI) and machine learning to extract:

- Text
- Handwriting
- Data

from scanned documents.

## **Functionality**

You upload a scanned document (e.g., driver's license), and Amazon Textract automatically analyzes it. The extracted information is then provided as a structured data file. This allows you to easily retrieve specific data points.

Textract can extract data from:

- Forms
- Tables
- PDFs
- Images

## **Use Cases**

Textract has various applications across different industries, including:

- **Financial Services:** Processing invoices and financial reports.
- **Healthcare:** Extracting information from medical records and insurance claims.
- **Public Sector:** Analyzing tax forms, ID documents, and passports.

## **Exam Relevance**

The key takeaway for the exam is that **Amazon Textract is the AWS service for extracting text, handwriting, and structured data from scanned documents using AI/ML.**
